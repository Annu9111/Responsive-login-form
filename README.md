# 🔐 Responsive Login Form

A clean and modern **Responsive Login Form UI** built using **HTML and CSS only**. This project focuses on creating a visually appealing and mobile-friendly login interface without using JavaScript.

---

## 💻 Live Demo

👉 https://responsive-login-form-nu.vercel.app/
---

## 🚀 Features

* 🎨 Modern UI with gradient background
* 📱 Fully responsive design
* 🔐 Login form with username & password fields
* ☑️ "Remember me" checkbox
* 🔗 Forgot password link
* 💡 Clean and minimal layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3
